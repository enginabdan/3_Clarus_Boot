										Data Visualization with Tableau
	
Covid-19 Tracking Dashboard
	1. Overview
		COVID-19 is the infectious disease caused by the coronavirus, SARS-CoV-2, which is a respiratory pathogen.
		World Health Organization first learned of this new virus from cases in Wuhan, Peopleâ€™s Republic of China on 31 December 2019.

	2. Objective
		This project requires you to explore given data set, design interactive dashboard,
		And then convey your results in a website through Google Sites. 

	3. Instructions
		There are two milestones in the project.
		aa. Creating Dashboard - First Milestone
 			1. Import the dataset using Web Data Connector.
			2. Go to Connect > To a Server > Web Data Connector
			3. Paste https://tableau.data.world/?dataset_name=covid-19-data-resource-hub%2Fcovid-19-case-counts&query=SELECT%20*%20FROM%20covid_19_activity&queryType=SQL in the box. 
			4. The dataset is a large dataset and may need extra computing time when creating views in Tableau
			5. Depending on your computer's hardware spesifications.
			6. In case you have problem to create the viz due to hardware,
			7. write a SQL query in the web connector page to return a specific date range or first n rows.
			8. We leave to you to decide on how many rows or which date range you will work.
			9. (SELECT * FROM covid_19_activity WHERE report_date BETWEEN '2021-06-01' AND '2021-08-31')
		bb. Creating a Google Site - Second Milestone
			1. Create a Google site.
			2. Choose a meaningful site name that reflects your site's purpose. 
			3. Please refer to the following link on how to create a google site:
			4. https://support.google.com/sites/answer/98081?hl=en
 			5. You can choose a background image from [unsplash.com](https://unsplash.com/) or any other source.
 			6. Embed the your interactive dashboard to your web site and publish.
